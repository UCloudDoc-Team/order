

# 订单字段定义 

| 字段     | 定义                                 | 示例                  |
|----------|--------------------------------------|-----------------------|
| 订单号   | 下单资源时生成的唯一编号             |                       |
| 创建时间 | 订单号生成的时间                     | 2025/11/4 23:00:02   |
| 开始时间 | 实际使用资源的时间                   | 2025/11/4 23:00:00   |
| 结束时间 | 终止使用资源的时间                   | 2025/11/5 0:00:00    |
| 订单类型 | 指订单的交易类型，包括如下类型：<br><div style="border: 2px solid blue; padding: 10px; border-radius: 5px; background: #f8f9fa; margin-top: 5px;"><br>• 新购：首次购买资源时生成的订单<br>• 续费：租约型预付费资源续费时产生的订单<br>• 升级：资源配置升级时的订单<br>• 降级：资源配置降级时的订单<br>• 结算：资源使用期限变更时产生的订单<br>• 补单：发现订单错误或者资源变更后人为的修改订单<br>• 删除：删除资源资源产生的欠费或者退费订单<br>• 过期：租约型预付费资源过期后按天生成的订单<br>• 后付费：后付费资源在每个付费周期结束时的结算订单<br>• 删除恢复：删除资源后恢复资源产生的订单<br></div> |          新购             |
| 支付状态 | 指订单的支付状态，包括如下状态：<br><br>• 待支付<br>• 已完成 | 已完成 |
| 资源ID | 云产品中用于唯一标识产品的标识符（短ID） | uhost-1itxvdze3kbn |
| 资源标识 | 资源的拓展信息 | 名称:UHost;内网IP:192.168.18.164; |
| 资源数量 | 用户购买的云产品数量 | 1 |
| 地域  | 云资源是分地区部署的，表示用户下单时选择的地域 | 乌兰察布 |
| 产品类型 | 用户购买的云产品的大类名称，如云主机 | 华北二可用区A |
| 当前配置 | 用户变更后的规格配置 | {{网络增强2.0:4档}}{{CPU快态}}:4核;{{内存快态}}:8192 MB;{{RSSD云盘系统盘}}:40 GB;}} |
| 历史配置 | 用户变更前的规格配置 | {{网络增强2.0:4档}}{{CPU快态}}:4核;{{内存快态}}:8192 MB;{{RSSD云盘系统盘}}:40 GB;}} |
| 项目名称 | 资源归属的项目，用户在控制台自主命名项目 | Default |
| 计费方式 | 指订单的收费方式，包括如下方式：<br><br>• 按年：用户以年为单位购买资源<br>• 按月：用户以月为单位购买资源<br>• 按天：用户以天为单位购买资源<br>• 按时：用户以时为单位购买资源<br>• 按量：根据云服务资源中实际计量收费<br>• 赠送：免费赠送一段时间的云资源使用<br>• 预支付-30天期：用户预先支付30天的资源费用<br>• 试用：试用期内免费提供给用户使用<br>• 后付费：用户先使用资源，在每个付费周期结束后再付费<br>• 抢占式：用户以大幅度低于市场的价格购买服务，但是服务商可以随时收回这些服务 |新购 |
| 交易量 | 资源使用的时长 | 1 |
| 订单金额 | 订单金额=代金券扣款+赠送金额扣款+现金账户扣款 | 28.66 |
